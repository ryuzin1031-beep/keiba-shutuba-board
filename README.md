# 重賞出馬表データボード

netkeiba の出馬表から重賞レースのデータを取得し、各出走馬の
種牡馬・騎手・調教師データと SABC 4段階評価(能力・距離適性・舞台適正・馬場適性)を
表示する静的サイト。

## 使い方

```bash
# 1. データ取得(race_id は netkeiba 出馬表URLの race_id= の値)
python3 scraper.py 202609030411

# 2. 表示
python3 -m http.server 8970 --directory .
# → http://localhost:8970 を開く
```

別の重賞を表示したい場合は、netkeiba の出馬表 URL
`https://race.netkeiba.com/race/shutuba.html?race_id=XXXXXXXXXXXX`
の race_id を指定して scraper.py を再実行するだけ。

## 機能

- 出馬表(枠・馬番・馬名・父・性齢・斤量・騎手・調教師)
- 行クリックで詳細パネル展開:
  - 本馬のコース実績(通算 / 芝全体 / 同競馬場 / 完全一致コース / 距離帯±200m)
  - 種牡馬の産駒成績(通算・当年、芝/ダ別、平均距離)
  - 騎手・調教師の通算・当年成績(勝率/連対率/複勝率、芝/ダ別)
  - 近5走
- SABC 評価: 過去成績から自動算出。バッジクリックで手動変更(S→A→B→C巡回)、
  localStorage にレース単位で保存。「リセット」ボタンで自動算出値に戻る。
  ※印はデータ不足による暫定評価。
- 並び替え: 馬番順 / 総合評価順 / 各評価項目順

## 評価ロジック(自動算出)

| 項目 | 根拠 |
|---|---|
| 能力 | 近5走平均着順 + G1勝ち・重賞好走実績 |
| 距離適性 | 同馬場・当該距離±200mでの複勝率 |
| 舞台適正 | 完全一致コース(なければ同競馬場同馬場)での複勝率 |
| 馬場適性 | 本馬の同馬場複勝率 + 父産駒の同馬場勝率の補正 |

複勝率 50%以上=S / 35%以上=A / 20%以上=B / 未満=C。
出走3走未満は S を保留して「※(データ少)」を付与。

## 制限事項

- 種牡馬の「競馬場別・コース別」産駒集計は netkeiba 無料領域では非公開(プレミアム機能)
  のため、産駒の芝/ダ別成績+平均距離と、本馬自身のコース実績で代替している。
- 騎手・調教師の競馬場別成績も同様に非公開のため、通算・当年・芝/ダ別を表示。
- netkeiba のページ構造変更で取得に失敗する場合がある(失敗項目は「データなし」表示)。
- スクレイピングは個人利用の範囲で。リクエスト間隔1秒、6時間のローカルキャッシュあり
  (`/tmp/keiba_cache`)。
